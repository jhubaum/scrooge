# Scrooge – A collection of scripts to manage and analyse your spendings 
named after the world's richest duck

![Scrooge McDuck](scrooge.png)

## Setup
- Clone this repository
- Create the config directory  
  Per default this is `~/.scrooge`, but you can override it with the `SCROOGE_CONFIG_PATH` environment variable.
- Copy `plan.py.template` to `SCROOGE_CONFIG_PATH/plan.py`
- Run the code via `python -m scrooge` from the repo's parent directory
